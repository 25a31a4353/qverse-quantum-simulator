# QVerse: Interactive Quantum Entanglement Lab

An immersive, real-time quantum circuit simulator and educational platform. QVerse allows users to build 2-qubit circuits, visualize state vectors on professional 3D Bloch spheres, and receive intelligent feedback from a context-aware AI Assistant.

## 🚀 Key Features
- **3D Bloch Sphere Visualization**: High-fidelity scientific rendering of qubit states using Three.js.
- **Interactive Circuit Builder**: Real-time gate application (H, X, CNOT) with immediate probability updates.
- **Quantum AI Assistant**: A context-aware guide that explains current simulation states and quantum physics concepts.
- **Entanglement Analytics**: Live tracking of Bell state generation and correlation strengths.

## 🛠️ Technology Stack
- **Frontend**: React, TypeScript, Vite
- **3D Engine**: Three.js, React Three Fiber, Drei
- **State Management**: Zustand
- **Animations**: Framer Motion
- **Visualization**: Recharts

## 💻 Getting Started
1. Clone the repository
2. Navigate to the project directory: `cd qverse`
3. Install dependencies: `npm install`
4. Run development server: `npm run dev`
